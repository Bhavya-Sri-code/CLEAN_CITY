Waste Management System
Project Overview

The Waste Management System is a web-based application built with HTML, CSS, and JavaScript. It is designed to help local municipalities and organizations manage waste collection efficiently. The system supports multiple user roles including Admin, Collector, and User, ensuring a smooth workflow for reporting, tracking, and managing waste disposal activities.

This project demonstrates front-end development skills, role-based access control, and dynamic web interfaces, making it an ideal showcase for internship or portfolio purposes.

Features
User Roles
Admin
Add, edit, or remove users (Collectors & Users).
Assign waste collection routes and tasks.
View overall waste collection reports and status.
Collector
Access assigned collection routes.
Update status of waste collected (Pending / Completed).
Communicate with Admin regarding collection issues.
User
Report waste collection requests in their area.
Track the status of their requests.
View notifications and updates regarding collection schedules.
General Features
Responsive Design: Works seamlessly on desktops, tablets, and mobile devices.
Interactive Dashboard: Real-time updates using JavaScript.
Role-based Access: Each user only accesses features relevant to their role.
Form Validation: Ensures accurate data input for all forms.
Intuitive UI/UX: Easy navigation for all users.
Technologies Used
