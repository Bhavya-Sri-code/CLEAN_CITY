Waste Management System
Overview

The Waste Management System is a web-based application built using HTML, CSS, and JavaScript. This system is designed to manage and streamline waste collection and disposal efficiently. It includes multiple user roles, allowing different types of users to interact with the system according to their responsibilities.

Features
User Roles:
Admin: Manage users, monitor waste collection activities, and assign tasks.
Collector: Update status of waste collection and track assigned routes.
User: Report waste issues and view collection updates.
Responsive UI: Works on both desktop and mobile devices.
Interactive Interface: Uses HTML forms and dynamic JavaScript for smooth user interactions.
Role-based Access: Each user type has a dedicated dashboard and access permissions.
Technologies Used
Frontend: HTML, CSS, JavaScript
Version Control: Git & GitHub
